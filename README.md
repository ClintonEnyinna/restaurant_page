Restaurant Landing Page
=======================

This is a restaurant landing page built purely with JavaScript. Webpack was used to bundle the javascripts files and other assets. 

The website has a menu, contact and about pages.

Features
--------
* Landing page
* Menu, Contact and About pages
* Built purely with JavaScript.

### Prerequisites

- css-loader "3.5.3"
- file-loader: "^6.0.0"
- style-loader: "1.2.1"
- webpack: "4.43.0"
